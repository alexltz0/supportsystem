In 4 Schritten starten

Einfach klonen, installieren und loslegen.

1

Abhängigkeiten installieren

$ npm install
2

Konfigurieren

Passe config.json (Branding + Datenbank) und public/css/colors.css (Farben) an.

// config.json
{
  "branding": {
    "name": "MeinProjekt",
    "subtitle": "Support-System",
    "logo": "img/logo.png"
  },
  "database": {
    "host": "localhost",
    "user": "root",
    "password": "1234",
    "database": "support",
    "connectionLimit": 10
  }
}
3

Farben anpassen (optional)

Ändere die Akzentfarben in public/css/colors.css — alles passt sich automatisch an.

/* public/css/colors.css */
:root {
  --accent: #ff0079;
  --accent-hover: #cc0062;
  --accent-light: #ff6fb5;
  /* ... weitere Farben ... */
}
4

Server starten

$ npm start
→ Server running at http://localhost:3000

MySQL muss laufen. Datenbank & Tabellen werden automatisch erstellt.
